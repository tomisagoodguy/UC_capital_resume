## 💸 小資族的主觀選股流程（量化後 + 主觀判斷）

1. 建立選股池
2. 檢查 K 線波動（過大、套牢、成交金額 > 1500 萬）
3. 營收爆發（轉盈或創新高） vs. 是否反映在股價
4. 檢查中實戶與散戶行為，觀察主力買入
5. 券商前 15 分點淨買超金額，每 5 日加總，觀察 8 週變化
6. 判斷融資使用率是否偏高（視為聰明錢）
